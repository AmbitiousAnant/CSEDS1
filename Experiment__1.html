COURSE CURRICULUM

B.Tech Computer Science Engineering (Data Science)

Academic Year: 2024-2025

This is the course curriculum for Bachelor of Technology in Computer Science & Engineering with specialization in Data Science. The program is designed for 4 years & includes theory as well as practical courses.

Department: CSE | University: AKTU | Copyright © 2024

SEMESTER I

First Semester Core Courses:

  1. CSE101 - Introduction to Programming - 4 Credits
  2. MTH101 - Engineering Mathematics-I - 4 Credits
  3. PHY101 - Engineering Physics - 3 Credits
  4. ENG101 - Communication Skills - 2 Credits
  5. ES101 - Basic Electrical Engineering - 3 Credits

SEMESTER II

Second Semester Core Courses:

  • CSE102 - Data Structures - 4 Credits
  • MTH102 - Engineering Mathematics-II - 4 Credits
  • CHM101 - Engineering Chemistry - 3 Credits
  • ES102 - Engineering Mechanics - 3 Credits
  • LAB102 - Programming Lab - 2 Credits

SEMESTER III

Third Semester Course Details:

CSE201 - Object Oriented Programming
This course covers OOP concepts using C++. Topics include classes, objects, inheritance & polymorphism. Credits: 4
CSE202 - Digital Logic Design
Study of logic gates, combinational & sequential circuits. Credits: 3
CSE203 - Computer Organization
Understanding of computer architecture & memory hierarchy. Credits: 3
MTH201 - Discrete Mathematics
Mathematical concepts including set theory & graph theory. Credits: 4
LAB201 - Data Structure Lab
Practical implementation of various data structures. Credits: 2

SEMESTER IV

Fourth Semester Course Structure:

  1. Theory Courses
    • CSE204 - Design Thinking - 4 Credits
      • Innovation Techniques
      • Problem Solving Methods
    • CSE205 - Operating Systems - 3 Credits
      • Process Management
      • Memory Management
      • File Systems
    • CSE206 - Database Management Systems - 3 Credits
      • SQL Queries
      • Normalization
      • Transaction Management
  2. Laboratory Courses
    • LAB204 - DBMS Lab - 2 Credits
    • LAB205 - OS Lab - 2 Credits
